## ⚙️ Installation

```bash
git clone https://github.com/TheNimaDev/Market-Price-Live.git
cd Market-Price-Live
npm install
```

## 🔗 Database Setup

```bash
npm run db
npm run mig
npm run seed
````

## ▶️ Run the Project

```bash
npm run dev
```
